FizzBuzz-BASH by anthony-looper

FizzBuzz, written in BASH Script.

This is my first attempt at FizzBuzz using BASH script. I had help and did a lot of reading online (http://www.it.uom.gr/teaching/linux/lsst/).

Dependencies

A Linux environment and permissions to make a script executable.

Install

Drop "fizzbuzz.sh" in Linux and make it executable.

License
